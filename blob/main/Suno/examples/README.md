
# ПРИМЕРЫ ПРОМПТОВ

## Назначение
Коллекция удачных промптов и экспериментов для генерации в Suno.

## Структура файлов
Каждый файл в этой папке содержит один или несколько связанных примеров с описанием использованных приёмов.

Формат названия: `название-приема.md` или `дата-описание.md`

## Как добавлять новые примеры
1. Создать файл с понятным названием
2. Внутри указать:
   - Цель примера
   - Полный промпт
   - Полученный результат
   - Какие приёмы сработали
3. Добавить строку в список выше
