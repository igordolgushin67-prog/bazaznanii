## О проекте
Структурированное руководство по созданию промптов для SUNO AI. Здесь собраны: базовая инструкция, библиотека проверенных приёмов, архивы текстов, опыта и треков.

## Быстрый старт
1. Начните с **[текущей версии инструкции](blob/main/Suno/versions/v3.2.md)**
2. Используйте **[библиотеку приёмов](blob/main/Suno/library/README.md)**
3. Изучайте примеры в **[архиве треков](blob/main/Suno/archive/треки/)**
4. Сохраняйте свои тексты в **[архив текстов](blob/main/Suno/archive/текст/)**
5. Фиксируйте наблюдения в **[архиве опыта](blob/main/Suno/archive/опыт/)**
