## О проекте
Это структурированное руководство по созданию промптов для SUNO AI. Здесь собраны: базовая инструкция, библиотека проверенных приемов, примеры из реальных треков и история версий.

## Быстрый старт
1. Начните с **[базовой инструкции](#базовая-инструкция-текущая-версия)** — там все основные правила.
2. Используйте **[библиотеку приемов](#библиотека-приемов-с-примерами)** — копируйте нужные синтаксисы.
3. Слушайте примеры по ссылкам — чтобы понимать, как прием звучит.
4. Экспериментируйте и комбинируйте.
