## О проекте
Структурированное руководство по созданию промптов для SUNO AI. Здесь собраны: базовая инструкция, библиотека проверенных приёмов с примерами из реальных треков, история версий, архивы созданных текстов и накопленного опыта.

## Быстрый старт
1. Начните с **[текущей версии инструкции](blob/main/Suno/versions/v3.2.md)** — там все основные правила.
2. Используйте **[библиотеку приёмов](blob/main/Suno/library/README.md)** — копируйте нужные синтаксисы.
3. Слушайте примеры по ссылкам в библиотеке и в **[архиве треков](blob/main/Suno/archive/треки/README.md)** — чтобы понимать, как приём звучит.
4. Экспериментируйте и комбинируйте.

## Как вносить новый опыт
1. После завершения диалога по команде "завершаем" создаются два файла: текст и опыт
2. Файлы сохраняются в соответствующие архивы с датой в названии
3. Новые треки добавляются в архив треков с указанием использованных приёмов
4. При накоплении значительных изменений создаётся новая версия инструкции
5. Все ссылки на приёмы ведут в библиотеку с примерами
