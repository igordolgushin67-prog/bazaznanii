- [2026-03-08-spoken-word-utochnenie.md](2026-03-08-spoken-word-utochnenie.md) — Уточнение по whispered и формату ссылок
- [2026-03-08-struktura-arkhivov.md](2026-03-08-struktura-arkhivov.md) — Оптимизация структуры архивов

## Содержание итогового опыта

### Ключевые выводы диалога
1. **Структура репозитория**: главный README → папки → в каждой папке свой README со списком файлов
2. **Формат ссылок**: три уровня (внутри README, короткий для копирования, полный для браузера)
3. **Завершение итерации**: три блока в отдельных окнах (текст, опыт, треки)
4. **Важные уточнения**: [Whispered] как команда, буква Ё, ударения ЗАГЛАВНЫМИ, распевы через дефис
5. **Оценка объёма**: ~500 сообщений, ~375 000 слов, диалог работает на сжатии

## Как добавлять новый опыт
При завершении итерации по команде "завершаем" новый опыт сохраняется в этой папке с датой в названии, а этот файл автоматически обновляется (добавляется одна строка в список).
