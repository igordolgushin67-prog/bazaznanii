# ИТОГОВЫЙ ОПЫТ ДИАЛОГА (08-09.03.2026)

## 1. СТРУКТУРА РЕПОЗИТОРИЯ
- Главный README.md ссылается только на папки (versions/, library/, archive/)
- В каждой папке свой README.md со списком файлов
- При добавлении нового файла обновляется только README в своей папке

## 2. ФОРМАТ ССЫЛОК
Три уровня:
- Внутри README: `папка/файл.md`
- Короткий для копирования: `blob/main/Suno/папка/файл.md`
- Полный для браузера: `https://github.com/.../blob/main/Suno/папка/файл.md`

## 3. ЗАВЕРШЕНИЕ ИТЕРАЦИИ
По команде "завершаем" создаются три блока в отдельных окнах:
- Текст: ссылка + содержимое
- Опыт: ссылка + содержимое
- Треки: ссылка + содержимое (если есть)

## 4. ВАЖНЫЕ УТОЧНЕНИЯ
- [Whispered] Текст... — команда, а не текст для пения
- Буква "Ё" критична (сЁстры, а не сестры)
- Ударения только ЗАГЛАВНЫМИ (гамАк, плитЫ)
- Распевы через дефис (те-енИ)
