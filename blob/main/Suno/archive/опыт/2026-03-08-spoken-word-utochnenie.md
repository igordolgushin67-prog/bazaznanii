# ОПЫТ ДИАЛОГА ОТ 08.03.2026

## 1. ВАЖНОЕ РАЗЛИЧИЕ: ИНСТРУКЦИЯ VS ТЕКСТ
- [Whispered] Текст... — команда Suno спеть эту конкретную строку шёпотом
- "тихий шёпот в небесах" как обычный текст — Suno воспримет как слова для пения, а не как инструкцию

## 2. ФОРМАТ ССЫЛОК
Для экономии времени пользователя ссылки выдавать в двух форматах:
- Короткий: blob/main/Suno/library/02-spoken-word.md
- Полный: https://github.com/.../blob/main/Suno/library/02-spoken-word.md

## 3. ЗАВЕРШЕНИЕ ИТЕРАЦИИ
По команде "завершаем" формировать два файла:
- Текст, над которым работали
- Опыт, накопленный в диалоге

## 4. УТОЧНЕНИЯ ПО РАБОТЕ С УДАРЕНИЯМИ
- Буква "Ё" важна (сЁстры, а не сестры)
- Ударения через ЗАГЛАВНЫЕ (гамАк, плитЫ, хлапОт)
- Распевы через дефис (те-енИ, фе-енИ)
