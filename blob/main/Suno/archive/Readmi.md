# Архив треков

В этой папке сохраняются списки сгенерированных треков с описанием использованных приёмов.

## Список файлов с треками
- [2026-03-08-full-list.md](2026-03-08-full-list.md) — Полный список треков от 08.03.2026

## Как добавлять новые треки
При появлении новых интересных треков создаётся новый файл с датой в названии, а этот файл автоматически обновляется.

## Как использовать
Каждый трек в списке содержит:
- Прямую ссылку на Suno
- Описание использованных приёмов
- Ссылки на соответствующие файлы в библиотеке
