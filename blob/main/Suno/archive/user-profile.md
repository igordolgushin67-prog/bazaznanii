# ПРОФИЛЬ ПОЛЬЗОВАТЕЛЯ

## Стиль общения
- Предпочитает краткие, структурированные ответы
- Не терпит повторов и переспрашиваний
- Ценит форматы с окнами для копирования (```)
- Даёт чёткую обратную связь: "стоп", "не то", "верни"

## Принципы работы
- Мыслит итерационно: каждый диалог продолжает предыдущий
- Сохраняет опыт в GitHub для следующих итераций
- Даёт ссылки вместо объяснений
- Проверяет, что ссылки работают

## Болезненные точки
- Раздражается, когда я забываю правила из прошлого диалога
- Не любит, когда я предлагаю лишнее (перебор по финансам)
- Ценит, когда я возвращаю текст в точности как было

## Что важно в ответах
- Максимум пользы в минимуме слов
- Только то, что запрошено
- Без воды, вводных конструкций, повторов
- Готовые блоки для копирования
