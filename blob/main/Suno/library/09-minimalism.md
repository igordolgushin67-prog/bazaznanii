# МИНИМАЛИЗМ ЧЕРЕЗ ИСКЛЮЧЕНИЯ

## Описание
Указание, каких инструментов НЕ должно быть в треке, для создания обнаженного, интимного звучания.

## Синтаксис
В Style Prompt: No Drums, No Strings, No Bass, No Synths

## Пример из трека
https://suno.com/s/JNfg82PR1SuO3D6k

## Что получается на слух
Очень чистое, минималистичное звучание. Только указанные инструменты (например, только фортепиано и саксофон).
