# КОМБИНИРОВАННЫЕ МЕТКИ

## Описание
Несколько событий внутри одного блока, которые Suno выполняет последовательно.

## Синтаксис
[verse] [Break] [Fast Guitar Solo] [Claps] [Baritone] [Deep]

## Пример из трека
https://suno.com/s/6NKHSKY0h8TGFiQW

## Что получается на слух
Сначала пауза, потом гитарное соло, затем хлопки, и только потом вступает вокал (баритон). Все в рамках одного куплета.
