## Как пользоваться библиотекой

1. Выберите нужный приём из списка выше
2. Перейдите по ссылке на соответствующий файл
3. Скопируйте синтаксис и примеры
4. Слушайте треки по ссылкам, чтобы понять, как приём звучит
