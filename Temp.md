# сюда можно вносить изменения
```
ПРАВИЛА ДЛЯ СЛЕДУЮЩЕЙ ИТЕРАЦИИ (на основе опыта)
```

1. ФОРМАТ ОТВЕТА:
   · Все готовые результаты помещаются в отдельные окна для копирования с обрамлением ```
   · При команде "финализируй" результат всегда разделять на два отдельных окна:
   · Первое окно: GENERAL STYLE (только стилистическое описание)
   · Второе окно: LYRICS (полный текст песни со всеми метками)
     · Внутри окон не должно быть заголовков, пояснений, отчетов — только контент
2. УПРАВЛЕНИЕ ДИАЛОГОМ:
   · При достижении 70% заполнения выводить предупреждение одной строкой
   · При достижении 85% автоматически выполнять команду "финализируй"
   · Команда "финализируй" = подготовка финальной версии + разделение на два окна
3. ВОКАЛЬНЫЕ МЕТКИ:
   · Формат: [Style: описание] или [голос: эмоция]
   · Для чистого русского произношения: [Style: pure Russian, warm, intimate]
   · Звуковые эффекты выносить отдельной строкой: [Sound of...]
   · Ад-либы отдельной строкой с меткой: [Male ad-lib: эмоция]
4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ К ТРЕКУ:
   · BPM: 128
   · Обязательные элементы: dance breaks, build-ups, anime interlude
   · Outro: чистый русский вокал без акцента
   · Все числа только словами (Восьмое марта, Восьмёрка)
5. СТРУКТУРА ТРЕКА:
   · Intro / Verse / Chorus / Bridge / Outro
   · Instrumental build-up перед ключевыми моментами
   · Dance break после энергичных частей
   · Pause для драматических пауз
6. ПРОВЕРКА ПЕРЕД ФИНАЛИЗАЦИЕЙ:
   · Отсутствие цифр
   · Отсутствие иностранного акцента в русских партиях
   · Корректность всех тегов в квадратных скобках
   · Четкое разделение вокальных партий
