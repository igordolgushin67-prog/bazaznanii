РАБОТА С ИНСТРУКЦИЕЙ:

При оформлении любого ответа, связанного с правилами Suno, всегда обращайся к этой инструкции. Если в ней отсутствует пункт, регулирующий текущий вопрос — добавь его в соответствующем месте с формулировкой, понятной для следующих итераций. Инструкция должна жить и развиваться.

ЗАПРЕТ НА ТАБЛИЦЫ:

Никогда не используй таблицы в финальных ответах — в копипасте они рассыпаются и становятся нечитаемыми. Вся информация должна подаваться либо связными абзацами, либо маркированными списками. Если нужно сравнить — используй два абзаца с подзаголовками «ЧТО ДЕЛАТЬ» и «ЧЕГО ИЗБЕГАТЬ».
1. ФОРМАТ ОТВЕТА:
   · Все готовые результаты помещаются в отдельные окна для копирования с обрамлением ```
   · При команде "своди" результат всегда разделять на два отдельных окна:
   · Первое окно: GENERAL STYLE (только стилистическое описание)
   · Второе окно: LYRICS (полный текст песни со всеми метками)
     · Внутри окон не должно быть заголовков, пояснений, отчетов — только контент
2. УПРАВЛЕНИЕ ДИАЛОГОМ:
   · При достижении 70% заполнения выводить предупреждение одной строкой
   · При достижении 85% автоматически выполнять команду "финализируй"
   · Команда "финализируй" = подготовка финальной версии + разделение на два окна
3. ВОКАЛЬНЫЕ МЕТКИ:
   · Формат: [Style: описание] или [голос: эмоция]
   · Для чистого русского произношения: [Style: pure Russian, warm, intimate]
   · Звуковые эффекты выносить отдельной строкой: [Sound of...]
   · Ад-либы отдельной строкой с меткой: [Male ad-lib: эмоция]
4. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ К ТРЕКУ:
   · BPM: 128
   · Обязательные элементы: dance breaks, build-ups, anime interlude
   · Outro: чистый русский вокал без акцента
   · Все числа только словами (Восьмое марта, Восьмёрка)
5. СТРУКТУРА ТРЕКА:
   · Intro / Verse / Chorus / Bridge / Outro
   · Instrumental build-up перед ключевыми моментами
   · Dance break после энергичных частей
   · Pause для драматических пауз
6. ПРОВЕРКА ПЕРЕД ФИНАЛИЗАЦИЕЙ:
   · Отсутствие цифр
   · Отсутствие иностранного акцента в русских партиях
   · Корректность всех тегов в квадратных скобках
   · Четкое разделение вокальных партий
Вот полный гайд по правилам оформления промптов для Suno AI (с учетом последних версий и лучших практик):

---

БАЗОВЫЕ ПРАВИЛА SUNO

1. СТРУКТУРНЫЕ ТЕГИ — ТОЛЬКО [ ]

Для обозначения частей песни используются только квадратные скобки:

```
[Intro]
[Verse 1]
[Pre-Chorus]
[Chorus]
[Post-Chorus]
[Bridge]
[Guitar Solo]
[Outro]
[Breakdown]
[Build-Up]
[Drop]
```

Важно: Текст внутри [ ] не поется, это инструкция для ИИ.

---

2. ОПИСАТЕЛЬНЫЕ ТЕГИ

Можно комбинировать внутри одних скобок через запятую:

```
[Intro, soft piano, atmospheric pads, female whisper]
[Guitar Solo, emotional electric guitar, soaring, like a cry]
[Bridge, stripped back, just vocals and piano]
[Chorus, explosive, full orchestra, gospel choir]
[Breakdown, electronic glitch, 808s, distorted vocals]
```

Правило: Описания должны быть на английском. Suno лучше понимает английские термины.

---

3. ВОКАЛЬНЫЕ УКАЗАНИЯ

Ставятся перед текстом, который нужно спеть особым образом:

```
[Whispered] Текст который шепчут...
[Screamed] Текст который кричат...
[Spoken word] А это просто говорят...
[Male vocals] Куплет поет мужчина...
[Female vocals, powerful] Женщина поет мощно...
[Duet, both voices] Вместе поют...
[Harmonies] Текст с гармониями...
[Rap delivery] А это рэп...
```

Можно комбинировать:

```
[Male vocals, emotional, tired] Наш этаж — королевство экранов и кода...
[Female vocals, belting, powerful] Дежурные — стражи консолей и тишины...
```

5. КРУГЛЫЕ СКОБКИ ( )

Используются для:

· Бэк-вокала (тихий второй голос)
· Эха/повторов
· Ад-либов (короткие вставки)

```
Ты моя звезда (моя звезда)
Я не забуду (никогда)
Спасибо (спасибо, спасибо...)
```

Suno воспринимает текст в ( ) как менее значимый, фоновый.

---

6. СТИЛЬ В НАЧАЛЕ (GENERAL STYLE)

Пишется одной строкой на английском, без скобок, в самом начале промпта:

Правильно:

```
GENERAL STYLE: Soulful House, Emotional Pop, Anthemic Electronic, 128 BPM, Major key, dramatic builds, male soulful tenor, female powerful gospel, piano intro, synth pads, guitar solo
```

Неправильно:

```
[GENERAL STYLE: Soulful House...]
(стиль: поп)
Стиль: рок с элементами...
```

---

7. ЧТО SUNO ИГНОРИРУЕТ

· Текст на русском внутри [ ] (если это не описание, а команда)
· Текст после # (комментарии)
· Лишние символы типа ~~~, *** между тегами
· Слишком длинные описания (>200 символов в одном теге)

9. ДОПОЛНИТЕЛЬНЫЕ СОВЕТЫ
ЧТО ДЕЛАТЬ:

Всегда используй квадратные скобки [ ] для инструкций и описаний. Пиши все описания на английском языке внутри скобок. Ставь вокальные теги непосредственно перед текстом, который нужно спеть особым образом. В самом начале промпта обязательно указывай GENERAL STYLE одной строкой на английском без каких-либо скобок. Экспериментируй с комбинациями стилей и описаний, чтобы найти уникальное звучание.

ЧЕГО ИЗБЕГАТЬ:

Никогда не используй круглые скобки ( ) для структурных тегов или основных инструкций. Не пиши описания на русском языке внутри квадратных скобок — Suno их не поймет. Не ставь вокальные указания после текста, только перед ним. Не разбивай GENERAL STYLE на несколько строк и не бери его в скобки. И главное — не копипасти чужие промпты без понимания, как они работают, всегда адаптируй под свою задачу.
