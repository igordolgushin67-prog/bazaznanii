ПРАВИЛА ОФОРМЛЕНИЯ ПРОМПТОВ ДЛЯ SUNO AI

1. СТРУКТУРНЫЕ ТЕГИ — ТОЛЬКО [ ]
Квадратные скобки для частей песни: [Intro], [Verse 1], [Chorus], [Bridge], [Outro], [Guitar Solo]. Текст внутри не поётся — это инструкция.

2. ОПИСАТЕЛЬНЫЕ ТЕГИ
Комбинируются внутри одних скобок через запятую: [Intro, soft piano, atmospheric pads]. Только английский язык.

3. ВОКАЛЬНЫЕ УКАЗАНИЯ
Ставятся перед текстом: [Whispered] текст..., [Male vocals, emotional] текст..., [Female vocals, powerful] текст..., [Duet] текст...

4. ЗВУКОВЫЕ ЭФФЕКТЫ
[crowd cheering], [applause], [static noise], [beep], [thunder], [silence]

5. КРУГЛЫЕ СКОБКИ ( )
Для бэк-вокала, эха, ад-либов: Спасибо (спасибо). Воспринимаются как фоновые.

6. GENERAL STYLE
Первая строка, одна строка на английском, без скобок. Пример: GENERAL STYLE: Soulful House, Emotional Pop, 128 BPM, male vocals, piano.

7. ЧТО SUNO ИГНОРИРУЕТ
Русский текст внутри [ ], текст после #, лишние символы ~~~ ***, слишком длинные описания.

8. ЗАПРЕТ НА ТАБЛИЦЫ
Таблицы не использовать — рассыпаются при копировании. Только абзацы или маркированные списки.

9. ОПТИМИЗАЦИЯ
Ответы максимально краткие без потери смысла. Без воды, вводных конструкций, повторов. Только суть.

10. РАБОТА С ИНСТРУКЦИЕЙ
При ответе обращаться к инструкции. Если нет нужного пункта — добавить.
